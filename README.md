# minesweeper-for-android

Приложение в RuStore: [Minesweeper for Android](https://apps.rustore.ru/app/com.ivanz851.minesweeper)

Сапер - игра-головоломка, в которой представлена сетка кликабельных клеток с скрытыми минами, разбросанными по игровому полю. Цель игры состоит в том, чтобы пометить
все клетки с минами и открыть все клетки, не содержащие мин. Игрок постепенно открывает
клетки (либо помечает их заминироватнными), используя подсказки о количестве соседних
мин в открытых клетках
